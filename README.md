# Tetris
This is a fully functional clone of the classic Tetris game, built using Python and Pygame.
I developed this project as a way to improve my understanding of game logic, object-oriented programming, and real-time input handling.

Features
 Classic Tetris shapes (I, O, T, S, Z, J, L) with rotation support

 Grid-based movement and collision detection

 Line-clearing mechanic and scoring system

 Real-time input handling with keyboard controls

 "Next Shape" preview window

 Full game loop with restart menu

 Clean UI with Pygame drawing and color rendering

 Technologies Used
 
Python 3
Pygame library
